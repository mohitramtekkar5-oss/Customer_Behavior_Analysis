### 🛠️ Data Pipeline & Methodology

* [cite_start]**Data Engineering (Python):** Cleaned missing review ratings using category-specific medians and engineered `age_group` features[cite: 19, 22].

* **Strategic SQL Analysis:**

### 💡 Key Business Insights

* [cite_start]**Target Demographics:** **Young Adults** are the highest revenue-contributing age group, generating **$62,143**[cite: 60, 65, 66].
* [cite_start]**Product Performance:** Top-performing categories include **Clothing** and **Accessories**, with items like **Blouse** and **Jewelry** leading in total orders[cite: 56, 57].
* [cite_start]**Shipping Trends:** Express shipping users show a higher average spend (**$60.48**) compared to those using Standard shipping (**$58.46**)[cite: 44, 46].
