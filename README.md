# 🛍️ Customer Shopping Behavior Analysis

This project provides an end-to-end data pipeline analyzing **3,900 customer transactions** to uncover spending patterns and drive strategic business decisions. By integrating **Python, SQL, and Power BI**, I transformed raw data into a dynamic intelligence dashboard

### 🛠️ Data Pipeline & Methodology

* **Data Engineering (Python):** Cleaned missing review ratings using category-specific medians and engineered `age_group` features.
* **Database Integration:** Connected the Python script to **PostgreSQL** to load the cleaned DataFrame for structured relational analysis.
* **Strategic SQL Analysis:**
    * **Revenue:** Identified Male customers as the primary revenue drivers at **$157,890** compared to **$75,191** from Female customers.
    * **Segmentation:** Classified the user base into **Loyal (3,116)**, **Returning (701)**, and **New (83)** segments.
    * **Subscription Impact:** Analyzed the **27% subscriber base** vs. non-subscribers to compare average spend and total revenue.
* **Business Intelligence (Power BI):** Developed an interactive dashboard visualizing a **$59.76 average purchase amount** and an average review rating of **3.75**.

### 💡 Key Business Insights

* **Target Demographics:** **Young Adults** are the highest revenue-contributing age group, generating **$62,143**.
* **Product Performance:** Top-performing categories include **Clothing** and **Accessories**, with items like **Blouse** and **Jewelry** leading in total orders.
* **Shipping Trends:** Express shipping users show a higher average spend (**$60.48**) compared to those using Standard shipping (**$58.46**).

### 🚀 Business Recommendations

* **Boost Subscriptions:** Promote exclusive benefits to the **73%** of customers who are currently non-subscribers.
* **Loyalty Programs:** Implement rewards specifically to transition the **701 Returning** customers into the **Loyal** segment.
* **Strategic Marketing:** Focus promotional campaigns on high-revenue segments like **Young Adults** and **Express-shipping** users.
